# Blacklist Command

## Syntax

```
/blacklist add [user | role | channel]
/blacklist remove [user | role | channel]
```

## Permissions

This command is available to users with the Manage Server permission.

## Description

The blacklist command allows you to blacklist users, roles, and channels from using the bot.
This is useful if you want to prevent certain users from using the bot, or if you want to prevent the bot from being used in certain channels.
