# Multiplier Command

## Syntax

```
/multiplier list
/multiplier clear
/multiplier set <type: Channel, Role, Server> [amount]
```

## Permissions

This command is available to users with the Manage Server permission.

## Description

The multiplier command allows you to set a multiplier for the amount of points a user earns in a channel, role, or server.
This multiplier is applied to the base amount of xp a user earns in a channel, role, or server. For example, if a user earns 3 xp from a message, and the multiplier for that message's channel is 2, the user will earn 6 points in that channel.
