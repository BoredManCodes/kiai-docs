# Rewards

Kiai has four different types of rewards:

-   Roles (both adding and removing)
-   Custom Messages
-   [Unbelievaboat](http://unbelievaboat.com) Currency
-   ~~API Webhook Event~~ Coming soon!

### Adding Rewards

To add a reward, use the [`/rewards add`](/commands/rewards) command.
Fill out the options and set the level for the reward there.

### Removing Rewards

To remove a reward, use the [`/rewards remove`](/commands/rewards) command.
The bot will present a list for you to choose from in the command option itself using Discord's autocomplete feature.

### Listing Rewards

To list all rewards, use the [`/rewards list`](/commands/rewards) command.
