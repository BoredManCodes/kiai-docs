### new KiaiClient()

> **new KiaiClient**(`apiKey`, `options`?): [`KiaiClient`](/api/KiaiClient)

Create a new KiaiClient

#### Parameters

| Parameter | Type | Description |
| ------ | ------ | ------ |
| `apiKey` | `string` | The API key to use for Kiai's API |
| `options`? | \{ `baseURL`: `string`; `debug`: `boolean`; `fetchFunction`: (`url`, `init`?) => [`Promise`](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Promise)\<[`Response`](https://developer.mozilla.org/docs/Web/API/Response)\>; `version`: `` `v${number}` ``; \} | The options to use to initialize the client |
| `options.baseURL`? | `string` | The base URL to use for Kiai's API |
| `options.debug`? | `boolean` | Whether to enable debug logging |
| `options.fetchFunction`? | (`url`, `init`?) => [`Promise`](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Promise)\<[`Response`](https://developer.mozilla.org/docs/Web/API/Response)\> | The fetch function to use for Kiai's API If you want to use your own fetch function (e.g. for a custom proxy), you can pass it here |
| `options.version`? | `` `v${number}` `` | The version of the API to use |

[`KiaiClient`](/api/KiaiClient)
